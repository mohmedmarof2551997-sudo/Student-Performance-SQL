# 🗄️ Student Performance — SQL Analysis

## 🎯 Objective
Analyze student performance data using SQL to extract behavioral and academic insights.

## 🛠️ Tech Stack
- SQL Server / T-SQL
- Aggregations · Filtering · Grouping
- ALTER TABLE · UPDATE · Feature Engineering

## 📌 Key Queries
| Query | Description |
|-------|-------------|
| AVG / MAX / MIN | Descriptive statistics on exam scores |
| GROUP BY sleep_hours | Exam score by sleep patterns |
| GROUP BY study_hours | Exam score by study habits |
| WHERE part_time_job | Compare students with/without jobs |
| Total_leisure_time | New engineered feature |
